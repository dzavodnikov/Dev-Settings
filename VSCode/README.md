# VSCode

This `settings.json` file contain common configurations and extensions configurations.

Common configurations applied by:

```sh
    $ mkdir -p                      ~/.config/Code/User/
    $ cp VSCode/settings.json       ~/.config/Code/User/
```

and synchronize back for common configurations:

```sh
    $ cp ~/.config/Code/User/settings.json ./VSCode/
```

> Note:
> On MacOS with Homebrew install `settings.json` located at `~/Library/Application\ Support/Code/User/`.
